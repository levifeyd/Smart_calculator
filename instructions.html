Инструкция по использованию Smart_Calc_v1.0

Установка:

  1. Вызвать цель "make install" из папки с  source-файлами.
  2. Дождаться установки *.app.

Удаление:

  1. Вызвать цель "make uninstall".
  2. Дождаться удаления  *.app.

Запуск и использование:

  1. Для запуска приложения открыть исполняемый файл test2.app, либо через терминал командой "open test2.app"
  2. Для вычисления: ввести выражение в поле с клавиатуры, либо кнопками и нажать на кнопку "=".
  3. Для построения графика: ввести выражение нажать  на кнопку "Построить график".

Запуск тестов:

  1. Для запуска тестов подсчета вызвать цель "make tests".
  2. Для создания html страницы с данными о покрытии функций тестами: вызвать цель "make gcov_report".
